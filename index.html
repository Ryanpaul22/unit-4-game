Crystal Collector


Instructions:

  1. You will be given a random "Number to Guess" at the start of each game.
  2. There are four crystals below. By clicking on a crystal, you will add a specific amount of points to the "Total Score".
  3. You win the game by matching the "Total Score" to the "Number to Guess", and you lose the game if the "Total Score" goes above the "Number to Guess"
  4. The value of each crystal is hidden from you until you click on it
  5. Each time a new game starts, the value of each of the crystals will change.
